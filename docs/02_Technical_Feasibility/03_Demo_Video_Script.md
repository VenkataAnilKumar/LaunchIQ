# Demo Video Script
## LaunchIQ — AI-Powered Product Launch Intelligence Platform

**Version:** 1.0
**Author:** Venkata Anil Kumar
**Date:** 2026-04-09
**Duration:** ~2 minutes

---

## Pre-Production Notes

- **Tone:** Confident, founder-led, energetic but focused
- **Format:** Screen recording with voiceover + optional face cam
- **Background music:** Subtle, upbeat, tech-forward
- **Screen:** Clean browser, dark mode preferred, no notifications visible

---

## Script

---

**[Screen: Blank slide or product logo animation]**

**[Screen: LaunchIQ logo appears]**

> "That's LaunchIQ."

**[Screen: LaunchIQ web app — clean dashboard, input form]**

**Voiceover:**

**[Screen: Typing into the input field]**
*Input text: "LaunchIQ is an AI-powered platform that helps startup founders and marketers create complete product launch strategies in minutes using autonomous AI agents."*

> "I hit Launch — and the agents go to work."

**[Screen: Orchestrator Agent activates — animated agent pipeline shows]**

### SCENE 3 — MARKET INTELLIGENCE AGENT (0:35–0:55)

**[Screen: Market Intelligence Agent panel — live streaming results]**

**Voiceover:**
> "First, the Market Intelligence Agent scans competitors, identifies trends, and surfaces market signals — in real time."

### SCENE 4 — AUDIENCE & STRATEGY AGENTS (0:55–1:20)

**Voiceover:**
> "The Audience Insight Agent builds buyer personas — with messaging angles for each segment."

> "Then the Launch Strategy Agent generates a full, phased go-to-market plan — tailored to my product and audience."

**[Screen: Strategy view — Phase 1 / 2 / 3 milestones with actions]**

> "This is a strategy that would normally take a consultant a week to produce."

### SCENE 5 — CONTENT GENERATION AGENT (1:20–1:40)

**[Screen: Content Generation Agent — copy streaming in]**
